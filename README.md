## 1. Наиболее близкое определение Активности это
- **отдельный автономный модуль функциональности приложения, который обычно напрямую связан с одним экраном пользовательского интерфейса и его соответствующими функциями.**
- Java-файл, в котом написан весь код Android-приложения.
- окно, в котором приложение рисует свой UI.

## 2. Приведенный фрагмент рисует
```java
paint.setStyle(Paint.Style.FILL);
paint.setColor(Color.GREEN);
canvas.drawRect(50, 80, 400, 280, paint);
paint.setColor(Color.YELLOW);
canvas.drawCircle(150, 150, 50, paint);
```
- зеленый круг в верхнем левом углу желтого прямоугольника
- ни один из вариантов
- желтый круг в нижнем правом углу зеленого прямоугольника
- **желтый круг в верхнем левом углу зеленого прямоугольника**

## 3. В результате выполнения приведенного фрагмента будет получен рисунок
```java
paint.setStyle(Paint.Style.FILL);
paint.setColor(Color.BLUE);
for (int i = 0; i < 5; i ++) {
  canvas.drawCircle(150, 150, i * 20, paint);
}
```
- **синий круг**
- четыре синих круга
- ни один из вариантов
- синяя окружность
